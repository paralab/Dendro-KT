Detailed Description

template<typename T, int dim>
class periodic::PCoord< T, dim >

Periodic integer coordinates glued by modulus operator. Specifies the period of the lattice in each axis. By default the period is the number of representable values, i.e. effectively non-periodic. If a finite period is specified, then each point is mapped to it's canonical representative upon construction.
