If there is a complete set of sibling leafs in a distributed sorted list, make sure they end up on the same rank. Makes it simpler to do mesh coarsening. Compare with Dendro-5.0 enforceSiblingsAreNotPartitioned(). In the below version, it is assumed that separated siblings might be partitioned across a consecutive sequence of two or more ranks.
